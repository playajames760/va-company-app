Operational Workflow
  1. Plan Flight: Create a Dispatch Release with routing + times.
  2. Load & Record Cargo (if applicable): Create Cargo Manifest referencing the flight.
  3. Execute Flight: After arrival, log actual times & cargo in the Crew Log.
  4. Issue/Update Notices: Add or cancel Company NOTAMs impacting operations.
  5. Maintain Fleet Data: Update Fleet entries when configuration or status changes.

Use action buttons on history pages or dynamic links below forms to move to the next step seamlessly.
