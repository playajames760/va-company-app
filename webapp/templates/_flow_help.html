Operational Workflow
  1. Plan Flight: Create a Dispatch Release with routing + times (edit later if plans change).
  2. Load & Record Cargo (if applicable): Create Cargo Manifest referencing the flight (auto links when possible).
  3. Execute Flight: After arrival, log actual times & cargo in the Crew Log (link to dispatch/cargo IDs).
  4. Issue/Update Notices: Add or cancel Company NOTAMs impacting operations.
  5. Maintain Fleet Data: Update Fleet entries when configuration or status changes.
